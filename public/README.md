# Photo drops

Drop two JPG files into this folder. They'll be served at the root of the site.

| File              | Used on        | Aspect       | Suggested resolution |
| ----------------- | -------------- | ------------ | -------------------- |
| `aton.jpg`        | Home hero      | 4:5 portrait | 1200 × 1500 px       |
| `aton-about.jpg`  | About hero     | 4:5 portrait | 1200 × 1500 px       |

Use a high-contrast headshot or a moody, slightly editorial portrait. Matte black + olive accent design — natural light or warm tungsten reads best.

If a file is missing, the portrait card falls back to a dark matte surface so the layout still holds.
